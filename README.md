# ASD (অটিজম স্পেকট্রাম ডিসঅর্ডার) স্ক্রিনিং টুল 🧠

এই প্রজেক্টটি অটিজম সচেতনতা বৃদ্ধি এবং প্রাথমিক স্ক্রিনিং সহায়তার উদ্দেশ্যে তৈরি একটি গবেষণামূলক উদ্যোগ। কৃত্রিম বুদ্ধিমত্তা (AI) ও মেশিন লার্নিং ব্যবহারের মাধ্যমে এটি বয়স অনুযায়ী বাংলায় সহজ ১০টি আচরণগত প্রশ্ন এবং মেডিকেল হিস্ট্রির ওপর ভিত্তি করে প্রাথমিক অটিজমের ঝুঁকি পরিমাপ করে।

## প্রধান ফীচারসমূহ ✨
- **বয়স-ভিত্তিক প্রশ্নমালা (AQ-10):** শিশু, বালক, কিশোর এবং প্রাপ্তবয়স্কদের জন্য আলাদা বাংলায় প্রশ্ন সেট।
- **স্বয়ংক্রিয় বয়স গ্রুপ সনাক্তকরণ:** বয়স ইনপুট করলেই গ্রুপ নির্বাচন হয়ে যায়।
- **করণীয় পদক্ষেপ ও থেরাপি পরামর্শ:** অটিজমের ঝুঁকি শনাক্ত হলে করণীয় পদক্ষেপসমূহ বিশদভাবে দেখানো হয়।
- **বিশেষজ্ঞ চিকিৎসক ও হাসপাতাল নির্দেশিকা:** বাংলাদেশের প্রধান হাসপাতাল, শিশু বিকাশ কেন্দ্র ও জরুরি হেল্পলাইন নাম্বারের তথ্য সরবরাহ করা হয়।
- **প্রিমিয়াম UI/UX ডিজাইন:** চমৎকার ডার্ক থিম, পার্টিকেল এনিমেশন, প্রোগ্রেস বার এবং রেসপন্সিভ লেআউট।

## কীভাবে রান করবেন (Local Run) 💻
১. রিপোজিটরি ক্লোন করুন এবং ডিপেন্ডেন্সি ইনস্টল করুন:
   ```bash
   pip install -r requirements.txt
   ```
২. ফ্লাস্ক সার্ভার রান করুন:
   ```bash
   python app.py
   ```
৩. ব্রাউজারে ওপেন করুন: `http://localhost:5000`
